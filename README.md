FactFeed

O FactFeed é um site informativo desenvolvido com o objetivo de conscientizar e educar as pessoas sobre as fake news, mostrando a importância da verificação de informações antes de compartilhá-las.

O projeto apresenta conteúdos educativos de forma simples e acessível, incentivando o pensamento crítico e o combate à desinformação.

Objetivo

O principal objetivo do FactFeed é informar os usuários sobre o que são fake news, como identificá-las e quais práticas podem ser adotadas para evitar a disseminação de informações falsas.

Tecnologias Utilizadas
HTML5
CSS3 (Grid Layout)
JavaScript (básico)
Funcionalidades
Página informativa sobre fake news.
Layout responsivo utilizando CSS Grid.
Interface simples e intuitiva.
Elementos interativos desenvolvidos com JavaScript.
